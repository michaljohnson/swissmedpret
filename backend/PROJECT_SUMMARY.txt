================================================================================
                              KEY FEATURES
================================================================================

REST API Endpoints:
  1. GET  /api/lexicon/search?term={keyword}
     - Search medical terms (German or English)
     - Returns matching entries with English translation and pictogram URLs

  2. GET  /api/languages
     - Get 20 supported languages (code and name)
     - Supports: German, English, French, Italian, Spanish, Portuguese, Dutch,
                 Polish, Russian, Arabic, Chinese, Japanese, Korean, Thai,
                 Turkish, Vietnamese, Indonesian, Hindi, Bengali, Punjabi

  3. POST /api/conversation/start
     - Initiate conversation session
     - Returns unique sessionId

  4. POST /api/conversation/stop
     - Terminate conversation session
     - Returns status confirmation

WebSocket Endpoint:
  - ws://localhost:8080/ws/conversation
  - Real-time translation with simulated 500-1500ms delay
  - Detects medical keywords from lexicon
  - Returns translation with detected keywords and timestamp

Medical Lexicon (15 Terms):
  Body Parts (7):    Handgelenk, Kopf, Herz, Lunge, Magen, Knie, Rücken
  Symptoms (3):      Fieber, Schmerzen, Übelkeit
  Diagnosis (2):     Infektion, Allergie
  Medications (2):   Tablette, Spritze
  Measurements (1):  Blutdruck
