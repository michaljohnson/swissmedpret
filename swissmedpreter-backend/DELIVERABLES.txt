                         QUICK START GUIDE
================================================================================

1. Build the project:
   cd /sessions/amazing-vibrant-euler/mnt/ase2_project/swissmedpreter-backend
   ./gradlew build

2. Run the application:
   ./gradlew bootRun

3. Test REST API (in another terminal):
   curl http://localhost:8080/api/languages

4. Test WebSocket (in browser console):
   const ws = new WebSocket('ws://localhost:8080/ws/conversation');
   ws.onopen = () => ws.send('{"text":"Test","sourceLang":"de","targetLang":"en"}');
   ws.onmessage = (e) => console.log(JSON.parse(e.data));
